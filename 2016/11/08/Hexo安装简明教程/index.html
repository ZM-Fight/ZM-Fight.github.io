问题

  • hexo默认的端口号是4000，但是4000端口如果被其他进程占用的话，这时访问localhost:4000会得不到响应。
    解决这个问题的方法是指定其他的端口hexo server -p 40000，访问localhost:40000
  • 问题描述：
    Hexo 3.0版本,在执行hexo deploy后出现报错，内容为：error deployer not found:github
    问题分析：
    Hexo3.0与以往版本最大的改变在于，更多的模块都从主程序中剥离了出来，其中就包括deploy的相关模块。
    解决方案：
    首先需要安装对应的deploy模块，目前Hexo支持以下服务器的一键部署：

安装命令：
npm install hexo-deployer-git --save //将git替换为别的名字就可以安装对应模块
